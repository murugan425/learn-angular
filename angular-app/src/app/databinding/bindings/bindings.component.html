Different way of Data Bindings

String Interpolation - To display data in the DOM
Event Binding - To handle events like click/inputs triggered by user
Property Binding - To update/assign variable to html element properties
Two Way Data Binding - To bind the data between the HTML/DOM and the typescript variables
Directive @Input - Data Binding - To bind the data in child component property by accessing it from the parent component
